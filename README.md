# Power BI Superstore Sales Dashboard

## Project Overview

This project is an interactive Power BI dashboard built using the Superstore dataset. The dashboard provides insights into sales performance, profitability, customer segments, regional trends, and top-performing products.

## Dashboard Preview

![Dashboard](dashboard.png)

## Key Features

* Sales performance analysis
* Profit and quantity tracking
* Regional sales comparison
* Top 10 products analysis
* Sales trend visualization
* Interactive slicers and filters
* KPI monitoring

## Tools Used

* Power BI
* DAX
* Data Visualization
* Excel Dataset

## KPIs Included

* Total Sales
* Total Profit
* Total Quantity
* Profit Margin

## Insights Generated

* Technology category generated the highest sales.
* West region showed strong performance.
* Consumer segment contributed the highest revenue.
* Sales demonstrated positive growth over time.
